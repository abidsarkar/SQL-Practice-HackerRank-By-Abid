## 🛠️ Skills Demonstrated

### Core SQL Capabilities
- **Filtering & Sorting:** `WHERE`, `ORDER BY`, `LIKE`, `BETWEEN`, `IN`
- **Aggregation:** `COUNT()`, `SUM()`, `AVG()`, `MAX()`, `MIN()`, `GROUP BY`, `HAVING`
- **Joins:** `INNER JOIN`, `LEFT JOIN`, `SELF JOIN`, multiple-table joins
- **Subqueries:** Correlated and non-correlated subqueries in `SELECT`, `WHERE`, and `FROM` clauses
- **Window Functions:** `ROW_NUMBER()`, `RANK()`, `LAG()`, `LEAD()`
- **String Functions:** `CONCAT()`, `SUBSTRING()`, `REPLACE()`
- **Numeric Functions:** `ROUND()`, `CEIL()`, `FLOOR()`, `ABS()`, `SQRT()`
- **Conditional Logic:** `CASE WHEN`, `COALESCE()`
- **Date Functions:** Extract, format, and calculate date differences

### Advanced Techniques
- Recursive Common Table Expressions (CTEs) for hierarchical/pattern queries
- Pivoting rows to columns using `CASE` aggregation
- Manhattan & Euclidean distance calculations
- Median calculation without built-in function
- Prime number generation using recursive logic
